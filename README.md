# Индивидуальный проект по теме "Генератор цитат и шуток"

- Ссылка на макет: https://www.figma.com/design/jBcV44fIDQQriyAqe2tpvx/%D0%98%D0%BD%D0%B4%D0%B8%D0%B2%D0%B8%D0%B4%D1%83%D0%B0%D0%BB%D1%8C%D0%BD%D1%8B%D0%B9-%D1%86%D0%B8%D1%84%D1%80%D0%BE%D0%B2%D0%BE%D0%B9-%D0%BF%D1%80%D0%BE%D0%B5%D0%BA%D1%82?node-id=0-1&t=YXlqM4hgX8J4AebZ-1

Стек: HTML, CSS, JS, SQLite

## base.js
Выполняет функцию хранилища данных и логики работы с ним.

Функции:
`initDatabase` - асинхронная функция, загружающая SQLite-базу в браузере и сохраняющая данные в массив.
`getRandomEntry` - получает случайную запись из массива.

## main.js
Выполняет функцию "моста" между HTML и данными.

Функции:
`setActiveButton` - визуально выдееляет выбранный фильтр. Добавляет класс активной кнопке и убирает у других.

## themeChanger.js
Выполняет функцию смену темы оформления сайта.
